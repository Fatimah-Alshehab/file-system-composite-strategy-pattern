# File System Composite & Strategy Pattern

⚠️ This project is part of a university assignment.

## 📌 Overview
This desktop application demonstrates the **Composite** and **Strategy Design Patterns** by representing and visualizing file and folder structures.

The program allows users to select a directory, recursively traverse its contents, build a hierarchical structure, calculate folder sizes, and display the results using different visualization strategies.

## 💻 Technologies
- C#
- Windows Forms (.NET)

## ⚙️ Features
- Select a folder from the system
- Recursively traverse files and subfolders
- Display:
  - Folder name
  - File name, size, and extension
- Calculate total size of folders using a single method call
- Visualize structure using different strategies:
  - **Tree Structure View** (hierarchical representation of files and folders)
  - **File Size Bar Chart** (displays size distribution of files and folders)

## 🧠 Design Patterns
- **Composite Pattern**: Represents files and folders as a unified tree structure
- **Strategy Pattern**: Allows switching between different visualization methods
